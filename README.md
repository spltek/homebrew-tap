# homebrew-tap

![Homebrew Tap](media/brew_tap_software_tools.png)

Homebrew tap for [spltek](https://github.com/spltek)'s tools.

This tap provides easy installation of SPLTek command-line utilities via Homebrew. The source code is available on GitHub: [github.com/spltek/homebrew-tap](https://github.com/spltek/homebrew-tap).

```
brew install spltek/tap/cl
```
